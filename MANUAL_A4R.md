# 🏁 MANUAL RÁPIDO - SISTEMA A4R SIMRACING

Este manual resume la puesta en marcha del sistema ServiRec en servidores remotos.

## 1. Acceso al Panel de Control (Web)
El administrador puede gestionar todo desde:
🔗 **URL:** [https://www.a4rsimracing.cl/a4r](https://www.a4rsimracing.cl/a4r)
*(Desde cualquier dispositivo con internet)*

---

## 2. Configuración de Simuladores (Notebooks)
Para cada simulador, debe seguir estos pasos:

1.  **Copiar Archivos**: Lleve el archivo `CyberLock.exe` y `config.ini` a una carpeta en la notebook.
2.  **Asignar ID**: Abra `config.ini` con el Bloc de Notas y cambie `simulador_id`:
    - Estación 01 -> `simulador_id = 1`
    - Estación 02 -> `simulador_id = 2`
    - *(Repetir según corresponda hasta la 6)*
3.  **Ejecutar**: Inicie `CyberLock.exe`.
    - **IMPORTANTE**: La primera vez, el sistema puede pedir permisos de Administrador. Acéptelos para que el bloqueo de pantalla funcione correctamente.

---

## 3. Funcionamiento Básico
-   **Bloqueo**: Al iniciar el PC, la pantalla se bloqueará automáticamente si no hay una sesión activa.
-   **Apertura**: Desde el panel web, elija la estación, asigne el tiempo y haga clic en **INICIAR**. El simulador se desbloqueará en menos de 10 segundos.
-   **Finalización**: Al agotarse el tiempo, la pantalla volverá a bloquearse automáticamente.
